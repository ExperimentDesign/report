## 3.2. User Stories.

**Epics**

| EPIC ID  | Título                                     | Descripción                                                                                                                               |
| :------- | :----------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------- |
| **EP01** | Accesibilidad de la Landing Page           | Como visitante, quiero que la información sobre la aplicación sea fácil de entender para comprender rápidamente su propósito.             |
| **EP02** | Interfaz de la Landing Page                | Como visitante, quiero que la página sea visualmente agradable para que atraiga mi atención.                                              |
| **EP03** | Acceso y Gestión de Roles                  | Como usuario, quiero acceder con mi información y que la aplicación me muestre distintas opciones según mi rol (Administrador o Docente). |
| **EP04** | Registro de Docentes                       | Como administrador, quiero registrar la información de los docentes de mi institución para que tengan acceso a la aplicación.             |
| **EP05** | Registro y Gestión de Aulas                | Como administrador, quiero registrar y gestionar las aulas de la institución para mantener la información actualizada.                    |
| **EP06** | Creación y Gestión de Recursos             | Como administrador, quiero crear y asignar recursos a las aulas para mantener un inventario actualizado.                                  |
| **EP07** | Creación y Gestión de Espacios Compartidos | Como administrador, quiero crear espacios compartidos (incluyendo deportivos) para que puedan ser gestionados y reservados.               |
| **EP08** | Registro y Gestión de Reuniones            | Como administrador, quiero registrar reuniones para que los docentes estén enterados de cuándo y dónde se realizarán.                     |
| **EP09** | Reserva de Espacios Compartidos            | Como docente, quiero reservar un espacio disponible para poder hacer uso del mismo en mis actividades académicas.                         |
| **EP10** | Consulta de Disponibilidad de Espacios     | Como docente, quiero consultar la disponibilidad de aulas y espacios para planificar mis actividades y futuras reservas.                  |
| **EP11** | Registro de Reportes de Averías            | Como docente, quiero registrar reportes sobre averías en los recursos de un aula para que la administración esté informada.               |
| **EP12** | Visualización de Historial de Reportes     | Como docente, quiero ver un historial de los reportes que he generado para hacer seguimiento a su estado.                                 |
